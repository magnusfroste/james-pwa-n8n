A Progressive Web Application for interacting with n8n workflows and AI agents. Built with React, TypeScript, and shadcn/ui for a modern, responsive chat interface.

## Features

- **PWA Support**: Install on any device with offline support
- **Chat Interface**: Modern chat interface for n8n AI agents
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Settings Management**: Configure your chat preferences
- **Modern UI**: Clean interface with shadcn/ui components

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **shadcn/ui** - Components
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **TanStack Query** - Data fetching
